# Orbitclash

Browser-based canvas action game with solo play, cooperative survival, and room-code PvP.

## Multiplayer

- Create or join with a six-character room code.
- Cooperative and PvP rooms support up to six players.
- PvP Clash supports automatic respawns and first-to-3, first-to-5, or first-to-10 KO matches.
- The room host synchronizes the battlefield, players, enemies, waves, health, score, and PvP standings.
- Encrypted WebRTC discovery connects browsers directly; the host enforces admission and the six-player capacity.

## Mobile

- Landscape-first two-thumb controls with fullscreen/orientation support.
- Compact phone layouts for the command menu, room lobby, HUD, and match report.
- Reduced particle, weather, visual-effect, DOM, and network update load on lower-power devices.
